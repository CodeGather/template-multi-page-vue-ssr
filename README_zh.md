## :gem: 使用方法

### 使用步骤

``` bash
# 安装依赖
npm install

# webpack-dev-server启动
npm run dev:page

# webpack打包构建
npm run build:page:dev   开发
npm run build:page:test  测试
npm run build:page:prod  生产
npm run serve:page:test  在本地模拟生产环境中启动服务
```
