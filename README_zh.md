一个基于 `Webpack` + `Vue` 的多页面服务端渲染项目模板

## 使用步骤

``` bash
# 安装依赖
npm install

# webpack-dev-server启动
npm run dev:client

# webpack打包构建
npm run build:dev   开发
npm run build:test  测试
npm run build:prod  生产
```
